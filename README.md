Auxclass:辅助函数操作类，函数基于ci框架
================
**Note** ：整理为gzq整理，相关函数也有其他的成果
#app相关操作函数，目录为app
 - Arrays：数组操作类
 - Common：公共操作类
 - Filter：过滤类
 - Regular：正则表达式类
 - Url：url处理类
 - UserAgent：头信息操作类
 - Version：app版本操作类
#缓存操作，目录为cache
 - Memcache：memcache操作类
 - Redis：redis操作类
#翻译操作：目录为translation
 - AppConfigOperate：地区、游戏相关翻译、礼包翻译、返回数据等
